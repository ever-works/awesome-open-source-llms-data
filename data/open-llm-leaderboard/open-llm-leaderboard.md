# Open LLM Leaderboard

## Overview

The Open LLM Leaderboard is a hub organization on Hugging Face dedicated to evaluating and comparing open-source large language models (LLMs). It provides a standardized platform for benchmarking LLMs across various tasks.

## Features

- Hosts datasets with detailed evaluation results and queries for models on the leaderboard
- Provides score results and current state of requests
- Enables users to look up model names for detailed predictions
- Maintains a transparent and open evaluation process

## Use Cases

- Researchers and developers can compare the performance of different open-source LLMs
- Users can identify top-performing models for specific tasks
- Track progress of LLM capabilities over time

## Pricing

Free to use; the leaderboard and associated datasets are openly accessible.

## Licensing

The leaderboard itself is provided by Hugging Face; model evaluations follow the licenses of the respective models.
