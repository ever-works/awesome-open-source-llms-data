## Overview

Groq is a hosted API provider specializing in ultra-fast inference for open source LLMs. It is optimized for real-time applications where low latency is critical.

## Features

- Fastest inference available among API providers
- Ideal for real-time applications requiring low latency
- Supports a selection of popular open source models
- Serverless hosted API with no infrastructure management
- Compatible with models available through API providers at competitive pricing ($0.20-0.80/M tokens)

## Pricing

Pay-per-use API pricing, estimated at $0.20-0.80 per million tokens depending on model selection.
