## Overview

Text Generation Inference (TGI) is an open source toolkit for deploying and serving large language models, developed by Hugging Face. It is designed for production workloads with optimized batching, continuous batching, and high throughput.

## Features

- Production-grade LLM serving
- Optimized throughput with continuous batching
- Native support for transformers models
- Tensor parallelism for multi-GPU setups
- Prometheus metrics for monitoring
- Token streaming support
- Safe tensors loading for fast startup
- Watermarking capabilities

## Deployment

- Designed for production-grade deployments
- Supports multi-GPU setups for large models
- Better throughput and batching compared to development-focused tools like Ollama
- Integrates well with Hugging Face model hub
