## Overview

LangGraph is a specialized agent framework within the LangChain ecosystem, released in 2024. It focuses on building controllable, stateful agents that maintain context throughout interactions. With over 24,800 GitHub stars and 34.5 million monthly downloads, it leads in enterprise adoption among agent frameworks.

## Core Capabilities

- Stateful agent orchestration with streaming support
- Support for single-agent, multi-agent, hierarchical, and sequential control flows
- Long-term memory and human-in-the-loop workflows
- Integration with LangChain products like LangSmith for monitoring agent performance

## Notable Enterprise Usage

- Klarna: Customer support bot handles two-thirds of all customer inquiries, doing the work of 853 employees and saving $60 million
- AppFolio: Copilot Realm-X improved response accuracy by 2x
- Elastic: AI-powered threat detection in SecOps tasks
- Additional users: Cisco, Uber, LinkedIn, BlackRock, and JPMorgan

## Best For

Enterprise applications requiring state management and complex agent orchestration workflows.
