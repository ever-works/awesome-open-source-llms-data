## Overview

TrueFoundry's AI Gateway provides a guardrail layer for enterprise LLM deployments, enabling precise policy enforcement and privacy-preserving transformations of inputs and outputs.

## Features

- **PII masking**: Automatically detects and anonymizes personally identifiable information (emails, SSNs, etc.) before it leaves the system
- **Topic filtering**: Block disallowed categories from appearing in model outputs
- **Custom word blocking**: Define and enforce lists of forbidden terms specific to the organization
- **Brand voice enforcement**: Ensure outputs align with company's communication guidelines
- **On-premises guardrail gateway**: Deployable within private infrastructure with no external data exposure
- **Configurable policies**: Administrators define precise rules tailored to business requirements

## Use Cases

- Enterprise applications handling sensitive customer data
- Organizations requiring strict data sovereignty and compliance
- Applications needing brand-safe output generation
