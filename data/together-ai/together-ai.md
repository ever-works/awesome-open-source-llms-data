## Overview

Together AI is a hosted API platform providing access to a wide selection of open source large language models with competitive pricing. It offers the benefits of open source models with the convenience of a SaaS deployment model.

## Features

- Wide selection of open source LLMs available via a single API
- Competitive pricing for inference
- Supports models including DeepSeek V3.2 Speciale, GLM-4.7, and MiMo-V2-Flash via API
- Typical API pricing at $0.20–0.80/M tokens
- Serverless deployment - no infrastructure to manage
- Suitable for production workloads

## Pricing

- Pay-per-use API pricing
- Estimated costs at $0.20–0.80 per million tokens depending on the model selected
