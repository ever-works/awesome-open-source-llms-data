## Overview

StarCoder2 is the next-generation code generation model from the BigCode project, building on the success of the original StarCoder. With improved architecture, enhanced training techniques, and better performance, StarCoder2 continues the mission of providing state-of-the-art open-source code models for the developer community.

## Architecture

- **Base:** Improved transformer architecture
- **Model Sizes:** Multiple variants (3B, 7B, 15B, and potentially larger)
- **Training:** Advanced techniques on expanded datasets
- **Languages:** 80+ programming languages
- **Context:** Extended context window
