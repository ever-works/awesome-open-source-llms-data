## Overview

Mistral Large 3 is a state-of-the-art open-weight model from Mistral AI trained with 41B active and 675B total parameters on 3000 NVIDIA H200 GPUs. It represents a significant advancement in open-source language model capabilities.

## Architecture

- **Total Parameters**: 675B
- **Active Parameters**: 41B per token
- **Architecture**: Mixture-of-Experts (MoE)
- **Training Hardware**: 3000 NVIDIA H200 GPUs

## Performance

Mistral Large 3 delivers 92% of GPT-5.2's performance at roughly 15% of the price, making it an extremely cost-effective option for enterprise deployments.

## Key Features

- State-of-the-art performance among open-weight models
- Efficient inference with MoE architecture
- Cost-effective alternative to proprietary models
- Strong multilingual capabilities
- Excellent reasoning and instruction-following

The MoE architecture enables efficient deployment despite the large total parameter count, as only 41B parameters are active per token.
