## Overview

Tülu 3 is an open source 405-billion-parameter large language model developed by the Allen Institute for AI (Ai2). It combines supervised fine-tuning (SFT) and reinforcement learning at a larger scale than previous iterations.

## Architecture

- 405 billion parameters
- Supervised fine-tuning combined with reinforcement learning
- Uses Reinforcement Learning from Verifiable Rewards (RLVR) framework

## Key Features

- Fine-tuned for tasks with verifiable outcomes
- Strong performance on mathematical problem-solving
- Improved instruction following capabilities
- Open source license enables research and commercial use

## Use Cases

- Mathematical reasoning and problem-solving
- Complex instruction following
- Research benchmarking
- Training pipeline development for verifiable reward systems
