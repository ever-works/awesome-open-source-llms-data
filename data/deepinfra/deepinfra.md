## Overview

DeepInfra is a hosted API platform providing cost-effective inference for open source large language models. It enables developers to access a wide selection of open models without managing infrastructure, with competitive pricing suited for batch workloads.

## Features

- Wide selection of open source LLMs available via API
- Competitive pricing, particularly cost-effective for batch workloads
- No infrastructure management required
- Supports models including DeepSeek V3.2 Speciale, GLM-4.7, MiMo-V2-Flash via API at $0.20–0.80/M tokens
- Serverless deployment model

## Pricing

- Pay-per-use pricing model
- API pricing estimated at $0.20–0.80 per million tokens depending on the model
