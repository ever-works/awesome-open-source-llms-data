## Overview

Seq2Seq (Sequence-to-Sequence) is a deep learning approach developed by Google for tasks including machine translation, image captioning, and natural language processing. It underlies several modern large language models.

## Architecture

- Uses a mix of encoders and decoders
- Encoder processes the input sequence into a fixed-size representation
- Decoder generates the output sequence from the encoded representation
- Decoder-only transformer variant used in models like LaMDA

## Legacy

- Underlies LaMDA (Language Model for Dialogue Applications)
- Underlies AlexaTM 20B, Amazon's large language model
- Foundation for many modern NLP architectures
- Paved the way for the transformer architecture that dominates current LLM designs
