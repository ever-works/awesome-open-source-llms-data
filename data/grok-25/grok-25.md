## Overview

Grok 2.5 is xAI's 270 billion parameter open-source model, made available to the community in August 2025. Originally trained in 2024, it now serves as a powerful baseline for developers and researchers exploring frontier-level reasoning capabilities.

## Model Architecture

- Parameters: 270 billion
- Architecture: Transformer-based with advanced training
- Training Data: Text-based tasks and reasoning
- Release: Open-source via Hugging Face in August 2025

## Capabilities

- Advanced reasoning on complex problems
- Real-time data integration
- Improved reasoning over previous versions
- Multi-step problem decomposition

## Deployment Requirements

- Hardware: 8 GPUs, each with minimum 40GB memory
- Model Size: ~500GB total
- File Count: 42 files
- Inference Engine: SGLang required

## Inference Setup

SGLang inference engine enables:
- Running an inference server
- Direct chat applications
- Flexible deployment scenarios
- Optimized token throughput

## Licensing

Weights covered by Grok 2 Community License Agreement:
- Model use and modification allowed
- Explicitly prohibits training other AI models
- Free for community and research use

## Future Plans

xAI confirmed plans to open-source Grok 3 approximately 6 months after Grok 2.5 release.
