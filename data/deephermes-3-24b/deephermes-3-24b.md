## Overview

DeepHermes-3 24B is Nous Research's Mistral-based variant of their toggle-on reasoning model. At 24 billion parameters, it provides a sweet spot between capability and efficiency.

## Model Specifications

- **Parameters**: 24 billion
- **Base**: Mistral 24B
- **Architecture**: Enhanced with reasoning tuning
- **Status**: Preview release

## Toggle-On Reasoning

- System prompt-based mode switching
- Extended thinking for complex tasks
- Fast response mode for simple queries
- Flexible reasoning depth

## Performance Profile

- Strong general capabilities
- Effective reasoning when enabled
- Good instruction following
- Balanced performance/speed

## Deployment Characteristics

- Single high-end GPU compatible
- Cloud and on-premises deployment
- Good inference speed
- Reasonable memory usage
- Production-ready
