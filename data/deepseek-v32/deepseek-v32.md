## Overview

DeepSeek V3.2 is a frontier-level open-source large language model featuring a Transformer architecture with DeepSeek Sparse Attention (DSA) MoE design. The model contains 685B total parameters and supports a maximum context length of 128K tokens.

## Architecture & Innovation

DeepSeek Sparse Attention is an efficient attention mechanism that substantially reduces computational complexity while preserving model performance, specifically optimized for long-context scenarios. This enables superior reasoning and agent performance through a scalable reinforcement learning framework.

## Performance Benchmarks

The model performs comparably to GPT-5 and the high-compute variant (V3.2-Speciale) surpasses GPT-5, exhibiting reasoning proficiency on par with Gemini-3.0-Pro. Achieved gold-medal performance in the 2025 International Mathematical Olympiad (IMO) and International Olympiad in Informatics (IOI).

## Key Capabilities

- Advanced reasoning for complex problem-solving
- Tool calling with "thinking with tools" capability
- Improved chat template format
- Agent capabilities for autonomous task execution
