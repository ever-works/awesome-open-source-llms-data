## Overview

Dify is a low-code platform for creating AI agents with a visual development interface that makes agent development accessible to non-technical users while still offering advanced capabilities for experienced developers. It has accumulated over 129,000 GitHub stars, the highest among the compared agent frameworks. The platform is distributed via Docker.

## Key Capabilities

- Low-code visual interface for agent development
- Built-in RAG (Retrieval-Augmented Generation), Function Calling, and ReAct strategies
- Support for hundreds of different LLMs
- TiDB Serverless vector search for scalability

## Use Cases

- Enterprise LLM gateway implementations
- Rapid prototyping for startups
- Document generation and financial report analysis
- Business workflow automation

## Best For

No-code and low-code agent development, enabling teams with varying technical expertise to build and deploy AI agents.
