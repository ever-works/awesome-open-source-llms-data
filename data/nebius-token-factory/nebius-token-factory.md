## Overview

Nebius Token Factory is a hosted API provider offering high-throughput inference for open source LLMs. The service runs on H100/H200 GPU clusters, providing competitive pricing for production workloads that require high throughput.

## Features

- High-throughput inference on H100/H200 GPU clusters
- Competitive pricing for production workloads
- Hosted API for open source models
- No infrastructure management required
- Suitable for production deployments

## Pricing

- Competitive usage-based pricing
- Pay-per-token model with volume discounts available
