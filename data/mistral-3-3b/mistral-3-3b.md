## Overview

Mistral 3 3B is the most compact model in the Mistral 3 family, delivering remarkable performance in a tiny package. Designed for maximum efficiency and minimal resource consumption.

## Use Cases

- Mobile and edge device deployment
- Real-time inference with minimal latency
- Resource-constrained environments
- Embedded systems
- On-device AI applications

## Model Variants

- **Base**: Foundation model
- **Instruct**: Instruction-tuned for dialogue
- **Reasoning**: Thinking mode for complex problems

## Capabilities

- Native image understanding despite small size
- Multilingual support across dozens of languages
- Reasonable reasoning performance for the size
- Efficient token generation

## Performance

Despite its compact size, matches or exceeds comparable proprietary models in its weight class. The instruct variant produces fewer tokens on average, improving throughput.

## Deployment

- Runs on smartphones and tablets
- Minimal memory footprint
- Sub-100ms inference latency possible
- Battery-efficient inference

## Quantization Support

Highly amenable to various quantization schemes for even smaller memory footprints while maintaining reasonable performance.
