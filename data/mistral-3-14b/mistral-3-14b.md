## Overview

Mistral 3 14B is a high-performing dense model that combines efficiency with capability. Part of the Mistral 3 family, it offers exceptional performance in reasoning and mathematical problem-solving.

## Model Variants

- **Base**: Foundation model for customization
- **Instruct**: Dialogue and instruction following
- **Reasoning**: Extended thinking for complex tasks

## Reasoning Capabilities

The reasoning variant achieves state-of-the-art accuracy for its weight class:
- AIME 2025: 85% accuracy
- Advanced mathematical problem-solving
- Complex logical reasoning
- Multi-step problem decomposition

## Performance Profile

Offers the best cost-to-performance ratio among open-source models, matching or exceeding comparable proprietary models while being significantly more efficient.

## Multimodal & Multilingual

- Native image understanding and analysis
- Support for dozens of languages
- Vision-language reasoning capabilities

## Deployment Options

- Local deployment in 24GB VRAM or less (FP8 quantization)
- Edge device support with quantization
- Cloud and on-premises deployment

## Technical Features

- Grouped-Query Attention for efficient inference
- Extended context support
- Low token generation overhead
