## Overview

Athene-V2 is Nexusflow's open-source 72B model suite, released as on-par alternatives to GPT-4o. Two specialized variants serve different use cases while maintaining exceptional performance across benchmarks.

## Model Variants

### Athene-V2-Chat-72B
- Focus: General chat, mathematics, coding
- Performance: On-par with GPT-4o
- Strength: Conversational excellence
- Ranking: Best open model on Chatbot Arena
- Task Coverage: Broad general-purpose

### Athene-V2-Agent-72B
- Focus: Function calling, agentic tasks
- Performance: Surpasses GPT-4o
- Strength: Complex tool use
- Specification: Superior at function calling
- Task Coverage: Agent orchestration

## Performance Benchmarks

- **Chatbot Arena**: Currently highest-ranked open-source model
- **Hard Reasoning**: Beats GPT-4o-0513
- **Mathematics**: Superior performance
- **Coding**: Excellent results
- **Function Calling**: Best-in-class (Agent variant)
- **Complex Workflows**: Advanced agentic capabilities

## Base Model

- Foundation: Qwen 2.5 72B
- Fine-tuning: Advanced instruction tuning
- Training: High-quality synthetic data
- Optimization: Specialized for each use case
