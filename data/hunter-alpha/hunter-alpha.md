## Overview

Hunter Alpha is currently the largest AI model made available for free, featuring 1 trillion parameters with approximately 32 billion active in its Mixture-of-Experts (MoE) architecture. It offers a 1 million token context window and is specifically optimized for agentic tasks, including multi-step workflows and tool usage. While its provider remains anonymous, it is hosted on OpenRouter.

## Features

- **Largest Available Model:** 1 trillion parameters (~32B active) for advanced processing capabilities.
- **Extensive Context Window:** 1 million token context window, ideal for long documents and complex interactions.
- **Agentic Optimization:** Designed for multi-step tasks and effective tool use.
- **Free Access:** Temporarily available for free, making it highly accessible for prototyping and research.
- **Web Search Integration:** Supports web search via the `:online` suffix.

## Limitations

- **Anonymity:** The model's provider is anonymous, lacking official documentation and benchmarks, which introduces uncertainty.
- **Data Logging:** All prompts are logged and may be used for training, making it unsuitable for sensitive data.
- **Local Execution:** Cannot be run locally, accessible only through the OpenRouter API.
