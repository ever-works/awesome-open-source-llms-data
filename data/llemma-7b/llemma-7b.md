## Overview

Llemma is a large language model specialized for mathematics, created by EleutherAI through continued pretraining of Code Llama on the Proof-Pile-2, a curated dataset of mathematical and scientific content.

## Model Variants

- **Llemma-7B**: Compact, efficient variant
- **Llemma-34B**: Larger, more capable variant

## Remarkable Performance

- **Key Achievement**: Llemma-7B outperforms Code Llama-34B on math benchmarks
- **Efficiency**: Demonstrates superior efficiency through specialization
- **Open Weights**: Complete transparency and reproducibility

## Training Data: Proof-Pile-2

- **Dataset Size**: 55 billion tokens
- **Content**: Scientific papers, mathematical websites, mathematical code
- **Coverage**: Diverse mathematical domains
- **Quality**: Curated for mathematical reasoning
- **Availability**: Open-source dataset
