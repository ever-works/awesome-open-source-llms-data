## Overview

PaliGemma is an open vision-language model from Google built on open components. It combines the SigLIP-So400m vision encoder with the Gemma-2B language model, creating a powerful tool for vision-language understanding.

## Architecture

- Image Encoder: SigLIP-So400m
- Text Decoder: Gemma-2B
- Design: Modular and open
- License: Open weights

## Core Capabilities

- **Image Captioning**: Generate descriptions from images
- **Visual Question Answering**: Answer questions about image content
- **Text in Images**: Extract and understand text within images
- **Object Detection**: Identify and locate objects
- **Image Segmentation**: Precise pixel-level understanding

## Fine-Tuning Performance

Demonstrates class-leading fine-tune performance across diverse vision-language tasks, making it ideal for:
- Custom domain adaptation
- Specialized use cases
