# Vijil Dome

Vijil Dome is a guardrails-as-a-service platform that provides a hosted guardrail API using fine-tuned LLMs and custom ML models to evaluate prompts and responses. It covers known jailbreak scenarios including 15+ DAN-style attacks and offers integration with monitoring tools like Weights & Biases.
