## Overview

Dolly is an instruction-following large language model from Databricks that demonstrated that high-quality instruction-tuning can be achieved with relatively small, curated datasets created by employees.

## Model Evolution

### Dolly 1.0
- Initial release demonstrating instruction-following
- Based on GPT-J-6B
- Trained on small curated dataset

### Dolly 2.0
- **Base Model**: EleutherAI's Pythia-12B
- **Training Data**: 15,000 instruction-response pairs
- **Data Source**: Human-generated by Databricks employees
- **Dataset**: Openly released as "databricks-dolly-15k"

## Key Innovation

**Small, High-Quality Dataset**:
Dolly demonstrated that instruction-following capabilities can be achieved with:
- Only 15K examples (vs millions for other models)
- Human-generated data
- Curated quality over quantity
- Employee-created instructions

## Comparison with Other Approaches

**vs Large-Scale Instruction Tuning**:
- Much smaller dataset (15K vs millions)
- Higher quality per example
- More accessible to reproduce
- Human-generated vs synthetic

**vs Few-Shot Learning**:
- Explicit instruction fine-tuning
- Better instruction-following
- More consistent outputs

## Limitations

Dolly acknowledges:
- Not state-of-the-art performance
- 12B parameter limitations
- Dataset size trade-offs
- Specific to tested categories
