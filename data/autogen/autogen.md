## Overview

AutoGen is a multi-agent conversation framework developed by Microsoft Research, released in September 2023. With over 54,600 GitHub stars, it has one of the largest GitHub followings among agent frameworks. The framework uses an event-driven architecture for complex interactions between AI agents.

## Key Features

- Multi-agent conversation framework with event-driven architecture
- Scalable workflows for complex collaborative tasks
- Extensive documentation with tutorials and migration guides
- Outperforms single-agent solutions on GAIA benchmarks

## Notable Usage

- Novo Nordisk implemented AutoGen for data science workflows
- Popular in data science and education sectors

## Current Status

In October 2025, Microsoft merged AutoGen with Semantic Kernel into the unified Microsoft Agent Framework, with general availability targeted for end of Q1 2026. AutoGen is now in maintenance mode, receiving only bug fixes and security patches, though existing projects continue to work.

## Best For

Complex multi-agent systems and data science workflows requiring collaborative agent interactions.
