## Overview

GLM-4-9B-0414 is a small-sized model in the GLM series with 9 billion parameters developed by THUDM. It inherits the technical characteristics of the larger GLM-4-32B series while offering a more lightweight deployment option ideal for resource-constrained edge scenarios.

## Architecture

- **Parameters**: 9 billion
- **Context Length**: 33K tokens
- **Subtype**: Chat
- **Series**: GLM-4 (derived from GLM-4-32B architecture)

## Capabilities

- Code generation
- Web design
- SVG graphics generation
- Search-based writing tasks
- Function calling support, allowing invocation of external tools to extend capabilities
- Strong benchmark performance for its size class

## Efficiency

- Inherits capabilities from the larger 32B model in a compact 9B package
- Good balance between efficiency and effectiveness in resource-constrained scenarios
- Optimized for edge deployment under limited computational resources

## Limitations

- Smaller context window (33K) compared to some competitors like Qwen3-8B (131K)
- Higher pricing at $0.086/M tokens on SiliconFlow

## Pricing

Available on SiliconFlow at $0.086/M tokens for both input and output.
