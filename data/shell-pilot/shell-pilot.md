## Overview
Shell-Pilot is a pure shell script tool that enables interaction with LLMs (Ollama, OpenAI, MistralAI) on Linux and macOS systems, facilitating intelligent system management without any external dependencies.

## Features
- Interact with Ollama, OpenAI, and MistralAI models
- Implemented purely in shell scripts, requiring no additional dependencies
- Compatible with Linux and macOS environments
- Enhances intelligent system management capabilities

## Pricing
Free and open-source.
